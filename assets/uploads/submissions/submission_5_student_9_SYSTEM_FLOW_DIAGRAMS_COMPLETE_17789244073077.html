📋 How Roles Interact:

Admin → Instructor: Creates instructor accounts, assigns to courses, broadcasts announcements

Admin → Student: Creates student accounts, enrolls in courses, broadcasts announcements

Instructor → Student: Creates lessons, assignments; records grades based on submissions

Student → Instructor: Views lessons, downloads materials, submits assignments for grading

Admin → Database: Full system analytics and monitoring access

Instructor → Database: Creates content, records grades, views analytics on own classes

Student → Database: Views content, submits work, views own grades

Key Interactions:
1. Admin Account Creation → Enables user login
2. Admin Course Creation → Determines what Instructors can teach & Students can enroll in
3. Instructor Lesson Creation → Content available to Students
4. Instructor Assignment Creation → Tasks assigned to Students
5. Student Assignment Submission → Triggers Instructor grading
